HOW TO GET STARTED
================================================================================

1. SETUP (5 Minutes)
   • Copy .env.example to .env
   • Set BOT_TOKEN (from @BotFather)
   • Set ADMIN_ID (from @userinfobot)
   • Run: docker-compose up -d

2. TEST (2 Minutes)
   • Open Telegram
   • Find your bot
   • Send: /start
   • Click buttons to test

3. CONFIGURE (10 Minutes)
   • Edit .env for additional settings
   • Set quality preferences
   • Configure cloud storage (optional)
   • Restart: docker-compose restart

4. DEPLOY (30 Minutes)
   • See DEPLOYMENT.md for production
   • Setup SSL with Let's Encrypt
   • Configure backups
   • Monitor logs

5. DEVELOP (Varies)
   • See DEVELOPMENT.md for adding features
   • Follow code style standards
   • Run tests before committing
